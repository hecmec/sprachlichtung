

## Ideen zum Kapitel "Einführung ins kritische Denken"

- Wir sollten in der Einleitung ansprechen, woher dann bitte Werte kommen.
- "Die Würde des Menschen ist unantastbar" Grundgesetz - Ist das eine Aussage? Woher wissen wir das? Warum ist das so? - Eine Frage der Wahl.

- Erklären, was das Overton Fenster ist und wie kritisches Denken uns hilft, es zu erkennen und zu hinterfragen.
