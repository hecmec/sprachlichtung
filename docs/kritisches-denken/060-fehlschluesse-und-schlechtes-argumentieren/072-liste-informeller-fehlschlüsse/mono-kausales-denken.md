:::tip Kurz
Eine komplexe Wirkung wird auf eine einzige Ursache zurückgeführt.

_Eine Ursache, wo viele zusammenwirken._
:::

## Definition

Beim **mono-kausalen Denken** (_causal oversimplification_) wird eine Wirkung, die in Wahrheit von **mehreren Ursachen** abhängt, auf eine **einzige** Ursache zurückgeführt. Das Zusammenspiel der Faktoren wird ausgeblendet.

Der Fehler: Viele Phänomene &mdash; gesellschaftliche, gesundheitliche, wirtschaftliche &mdash; entstehen aus einem Geflecht von Ursachen. Wer nur eine herausgreift, vereinfacht unzulässig und zieht oft die falschen Schlüsse.

Der Fehlschluss hat die folgende Form:

1. Die Wirkung W hat die Ursachen U₁, U₂, U₃ …
2. W wird allein auf U₁ zurückgeführt.
3. Die übrigen Ursachen werden ignoriert.

EN: _causal oversimplification_

## Verwandtschaft

<!-- zähle ähnliche Fehlschlüsse auf -->

- **Post hoc ergo propter hoc** &mdash; verwandt: vorschnelle Ursachenzuschreibung.
- **Falsches Dilemma** &mdash; verwandt: künstliche Verengung der Möglichkeiten.
- **Rosinenpicken** &mdash; verwandt, wenn nur der eine passende Faktor betont wird.

## Beispiele

### Beispiel 1

<!-- Beschreibung Beispiel eins -->

> „Seit man die Prügelstrafe abgeschafft hat, fehlt den Kindern jede Disziplin.“

Disziplin hängt von vielen Faktoren ab (Erziehung, Vorbilder, Schule, soziales Umfeld). Sie allein an einer Massnahme festzumachen, ist eine unzulässige Vereinfachung.

### Beispiel 2

<!-- Beschreibung Beispiel zwei -->

> „Er ist arm, weil er einfach zu faul ist.“

Armut hat in der Regel viele Ursachen (Bildung, Gesundheit, Arbeitsmarkt, Herkunft). Eine einzige herauszugreifen, verzeichnet das Bild.
