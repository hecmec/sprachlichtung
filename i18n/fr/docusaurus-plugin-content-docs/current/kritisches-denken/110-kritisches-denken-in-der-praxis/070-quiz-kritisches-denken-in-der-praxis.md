- b) La collecte d'informations

- a) Le tableau pour-contre
- b) L'analyse SWOT
- c) La cartographie d'arguments

- c) Les chambres d'écho et les bulles de filtres

<!--t src=b328fcee-->

- b) Les objectifs devraient être spécifiques et mesurables
- c) Les objectifs devraient être fixés par d'autres
- d) Les objectifs devraient se concentrer sur les connaissances théoriques

5. b) Les objectifs devraient être spécifiques et mesurables
